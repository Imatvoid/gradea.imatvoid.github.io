Maven 踩坑 -- java.lang.OutOfMemoryError

2016-02-18

java.lang.OutOfMemoryError: Java heap space

maven使用的堆内存溢出了,调大它的值即可.

在idea中

file->setting->Build,Execution,Deployment->Maven->Runner
VM option栏设置jvm参数:-Xms1024m -Xmx1024m -Xss1m

使用命令行

在环境变量中加上
export MAVEN_OPTS="-Xms1024m -Xmx1024m -Xss1m"
linux下比如在~/.bash_profile中加入,然后新开个shell.

附录

再比如解决自动热部署时java.lang.OutOfMemoryError: PermGen space问题
需要增大PermGen区，默认为 64m，设置方法为：
-XX:MaxPermSize=128M
jdk8+后移除了PermGen,使用元数据区实现,对应的设置项为:
-XX:MaxMetaspaceSize=128m
