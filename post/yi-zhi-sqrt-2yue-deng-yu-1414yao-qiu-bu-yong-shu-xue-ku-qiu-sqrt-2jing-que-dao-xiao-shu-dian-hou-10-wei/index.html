已知 sqrt (2)约等于 1.414，要求不用数学库，求 sqrt (2)精确到小数点后 10 位

问题

已知 sqrt (2)约等于 1.414，要求不用数学库，求 sqrt (2)精确到小数点后 10 位

出题人

阿里巴巴出题专家：文景／阿里云 CDN 资深技术专家

考察点

  1. 基础算法的灵活应用能力（二分法学过数据结构的同学都知道，但不一定往这个方向考虑；如果学过数值计算的同学，应该还要能想到牛顿迭代法并解释清楚）
  2. 退出条件设计

解法

二分查找法
当high-low的差距小于 0.0000000001,那区间里任意取一个值与精确值的差距不会大于 0.0000000001
这也就是精确十位数字.


    public static double sqrt2() {
        double EPSINON = 0.0000000001;
        double low = 1.4f;
        double high = 1.5f;
        double mid;
        while (true) {

            mid = (low + high) / 2;
            if (high - low < EPSINON) {
                break;
            }
            if (high * low < 2) {
                low = mid;
            } else {
                high = mid;
            }
        }
        return high;


    }

    public static void main(String[] args) {
        //mid = 1.4142135623737027
        System.out.println(sqrt2());
        System.out.println(Math.sqrt(2));
    }
